# gigi-worker

The Gigi mobile app. React Native + Expo + TypeScript + Expo Router.

## Run it (development)

```bash
npm install
npx expo start
```

A QR code prints in the terminal. Open **Expo Go** on your iPhone, scan the QR, and the Gigi app opens on your phone.

Your Mac and phone must be on the same wifi network.

## Screens

- `app/index.tsx` — Welcome screen
- `app/scan.tsx` — QR scan placeholder (simulates a scan)
- `app/candidate.tsx` — Venue-view demo (Greg as a candidate, 4 action buttons, paid-feature locks)
