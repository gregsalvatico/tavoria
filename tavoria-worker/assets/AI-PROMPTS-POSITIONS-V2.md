# Position Icons — Modern Flat Illustration Style

Replaces the painterly Italian style with a clean, modern, editorial flat illustration look that matches the rest of the Gigi UI (bold colors, soft shadows, premium app feel).

**Style spec:**
- Flat / soft-3D editorial illustration style (think Stripe / Notion / Linear / Apple's recent app icons)
- Bold solid color backgrounds with subtle gradient depth
- One iconic object per position — no people, no faces
- Clean minimal shapes, gentle drop shadows, subtle highlights for soft 3D feel
- Limited palette per icon (1 dominant + 1-2 accents)
- Brand orange (`#FF5A1F`) used sparingly as an accent where it fits
- Bright modern energy — Italian hospitality without the painterly nostalgia
- Square 1:1, 1024×1024

**Replace filenames:** `position-barista.png`, `position-waiter.png`, `position-runner.png`, `position-cashier.png`, `position-host.png`, `position-bartender.png`, `position-cook.png`, `position-chef.png`, `position-cleaner.png` in this `assets/` folder.

## After you generate them

Save each PNG with the exact filename listed above. They'll overwrite the existing painterly versions, and the app will pick them up on next reload — code already wired to these filenames.

## Tool recommendations

- **ChatGPT (GPT-4o / DALL-E)** — strongest for flat editorial illustration style
- **Midjourney** — try `--style raw --ar 1:1` plus add "vector illustration" or "flat editorial illustration" 
- **Ideogram** — clean and crisp output
- For consistency, after generating the first one, attach it as a style reference for the next 8
